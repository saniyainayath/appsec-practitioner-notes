# Certified Application Security Practitioner (CAP) Study Notes

These notes were created while preparing for the **Certified Application Security Practitioner (CAP)** certification.

The goal of this repository is to organize important **application security concepts, OWASP Top 10 vulnerabilities, and secure coding principles** in a structured and easy-to-understand format.
